The Database contains the following tables for this module:

rewards_currency
rewards_customer
rewards_store_currency
rewards_transfer
rewards_transfer_reference

As well, salesrule table has been appended with the following columns:
`points_action`							VARCHAR
`points_currency_id`					INTEGER
`points_amount`							INTEGER
`points_amount_step`					FLOAT		(PRECISION 2, DEFAULT 1.00)
`points_amount_step_currency_id`		INTEGER		(NOT NULL)
`points_qty_step`						INTEGER		(NOT NULL, DEFAULT 1)
`points_max_qty`						INTEGER
